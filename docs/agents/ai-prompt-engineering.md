# AI Prompting & Safety (summary)

Key takeaways from `.github/instructions/ai-prompt-engineering-safety-best-practices.instructions.md`:

- Document prompt intent, limitations, and expected output formats.
- Avoid inserting sensitive data into prompts; sanitize inputs.
- Write prompts that are explicit, unambiguous, and localized-friendly (avoid idioms).
- Record reasoning and mitigation for potential biases; include tests or validation for AI outputs when applicable.
